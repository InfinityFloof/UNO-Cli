# UNO-Cli
## What is it?
UNO-Cli is a work in progress UNO clone written in Go designed to run in a CLI enviroment

## Dependencies
```
make
go
```

## Installation
```bash
make
make install
``` 

## Uninstallation
```bash
make uninstall
```
## Binaries
Binaries exist if you don't want to compile it yourself

## Usage
Write ``uno`` to start a game
